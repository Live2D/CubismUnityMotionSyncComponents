# お知らせ

## [制限事項] Cubism SDK MotionSync Plugin for Unity の対応環境について (2023-08-17)

本プラグインのバージョン R1 beta1 (5-r.1-beta.1) 時点では以下の環境のみ動作を確認しております。

* Windows
  * x86
  * x86_64
* macOS
  * x86_64
  * ARM64
 
以下の環境については、順次対応を行います。

* Android
* iOS
* WebGL
